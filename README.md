📄 100 Lawyers — Legal Services Website

100 Lawyers is a responsive and production-ready website for a legal services company, built with a focus on scalability, maintainability, and clean architecture. The project demonstrates a structured approach to layout development and styling using modern HTML and CSS standards.

🚀 Key Features

Responsive layout with a mobile-first approach

Semantic HTML5 structure for accessibility and SEO

Modular and reusable CSS architecture (BEM methodology)

Layout implementation using Flexbox and CSS Grid

Well-organized file structure for scalability

Cross-browser compatibility and consistent rendering

Optimized UI with clear visual hierarchy and typography system

🧩 Architecture & Approach

BEM (Block Element Modifier) naming convention for maintainable styles

Separation of concerns between structure (HTML) and presentation (CSS)

Component-based layout structure (header, hero, services, team, contact, footer)

Reusable UI patterns and consistent spacing system

🛠️ Tech Stack

HTML5 — semantic, accessible markup

CSS3 — advanced styling, layout, and responsiveness (Flexbox, Grid)

🎯 Project Goals

The main goal of this project is to showcase practical frontend development skills:

Building scalable and maintainable layouts

Applying modern CSS methodologies

Creating responsive interfaces without JavaScript

📌 Use Case

This project can serve as a solid template for legal companies, consulting agencies, or other professional services that require a clean and trustworthy web presence.
